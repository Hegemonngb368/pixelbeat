This guide is for Windows users who want the fastest path from download to playback.

### 1. Open the download page

### 2. Get the app

On the project page, look for the latest release or the main download files. Download the Windows version if one is listed. If the project provides a ZIP file, save it to your computer and unzip it into a folder you can find later, such as `Downloads` or `Desktop`.

### 4. Allow music access

If pixelbeat asks for a music folder, choose a folder on your PC that has audio files in it. Common file types are:

- FLAC
- WAV
- M4A
- OGG

Use the on-screen keys or menu hints to play, pause, skip, or change tracks. The app is built for quick use while you work, so most controls should stay close at hand.
